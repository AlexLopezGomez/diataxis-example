# Bienvenido

Esta es la documentación oficial de nuestra API.

## Primeros pasos

1. Obtén tu API key
2. Configura la autenticación
3. Realiza tu primera llamada

## Base URL

```bash
https://api.tudominio.com/v1
```

## Autenticación

Incluye tu API key en el header de cada petición:

```bash
Authorization: Bearer TU_API_KEY
```
